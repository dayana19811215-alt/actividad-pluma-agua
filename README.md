## Descripción

Este proyecto fue desarrollado en Python como parte de la actividad "La Pluma de Agua".

El programa calcula el caudal total, el consumo diario y el consumo mensual de agua a partir del número de plumas asignadas a un hogar.

Además, clasifica el nivel de consumo en bajo, medio o alto.

## Entradas

- Número de plumas de agua asignadas al hogar.

## Procesos

- Conversión de plumas a caudal total.
- Cálculo del consumo diario.
- Cálculo del consumo mensual.
- Clasificación del nivel de consumo.

## Salidas

- Caudal total (L/s).
- Consumo diario (litros).
- Consumo mensual (litros).
- Nivel de consumo.

## Herramientas utilizadas

- Python
- Google Colab
- GitHub

## Objetivo

Aplicar conceptos básicos de programación para modelar un problema relacionado con el consumo de agua, integrando elementos históricos, culturales e ingenieriles.
